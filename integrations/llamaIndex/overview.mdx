You can use the Nebius AI Studio third-party integration with [LlamaIndex](https://www.llamaindex.ai/). This is a Python framework that provides high-level libraries and enables you to create applications and agents based on AI models. If you have already created a LlamaIndex-based application, you can embed requests to Nebius AI Studio into your application.

The integration supports the following types of models:
