In addition to the [playground](https://docs.nebius.com/studio/inference/playground) and the OpenAI-compatible [API](https://docs.nebius.com/studio/inference/api), Nebius AI Studio provides integrations with multiple third-party instruments for inference. If you already use some of these instruments for your inference tasks, you can set them up to work with Nebius AI Studio.
